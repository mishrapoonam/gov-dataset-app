# ONS Dataset Explorer

Browse Office for National Statistics datasets using GOV.UK Design System.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16 (App Router)
- GOV.UK Design System
- TypeScript
- ONS API

## Project Structure

```
├── app/           # Pages
├── components/    # React components
├── lib/          # API and types
└── public/       # Static assets
```
